폰트 관련 속성

손흥민(32·LA FC)이 미국 메이저리그사커(MLS) 선발 데뷔전에서 두 골을 도우면서 경기 공식 최우수선수(POM·플레이어 오브 더 매치)에 뽑혔다. MLS 홈페이지도 손흥민의 활약을 ‘머리기사’로 다뤘다. 다만 기록 정정으로 공식 기록에는 도움 한 개만 남았다.
